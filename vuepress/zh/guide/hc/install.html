# 安装

前提条件

需要依赖 用户脚本管理器(推荐Tampermonkey)，如果已有可忽略下一步，直接安装划词搜题脚本

# 安装脚本管理器

要使用用户脚本，您首先需要安装一个用户脚本管理器。您可以根据当前使用的浏览器来选择一个用户脚本管理器。

桌面端

  • Chrome：Tampermonkey (opens new window) 或 Violentmonkey (opens new window)
  • Firefox： Greasemonkey (opens new window) 、Tampermonkey (opens new window) 或 Violentmonkey (opens new window)
  • Safari：Tampermonkey (opens new window) 或 Userscripts (opens new window)
  • Microsoft Edge：Tampermonkey (opens new window)
  • Opera：Tampermonkey (opens new window) 或 Violentmonkey (opens new window)
  • Maxthon：Violentmonkey (opens new window)
  • AdGuard：（不需要其他软件）

手机端(Android)

  • Firefox：Greasemonkey (opens new window) 、Tampermonkey (opens new window) 或 Violentmonkey (opens new window)
  • Maxthon：Violentmonkey (opens new window)
  • Dolphin：Tampermonkey (opens new window)
  • UC：Tampermonkey (opens new window)
  • Kiwi：Tampermonkey (opens new window) 或 Violentmonkey (opens new window)
  • X浏览器
  • 书签地球 (opens new window)
  • M浏览器 (opens new window)

手机端(iOS)

  • Safari：Tampermonkey (opens new window) 或 Userscripts (opens new window)
  • Gear (opens new window) ：（不需要其他软件）

# 安装划词搜题脚本

  • 找脚本官网 安装链接 (opens new window)

第一步：点击用户脚本页面上绿色的直接安装按钮，您的用户脚本管理器将向您确认是否安装。

第二步：点击安装即可
